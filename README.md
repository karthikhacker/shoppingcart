# shoppingcart

Shopping cart using MERN stack
# Running mern app
* Running mern app 
* Clone this repo using git clone git@github.com:karthikhacker/shoppingcart.git 
* cd into cd shoppingcart
* Run sudo npm install
* cd into cd frontend
* Run sudo npm install
* Then run cd ..
* Back to main app directory
* Then run npm run dev
